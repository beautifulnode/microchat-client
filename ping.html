Load Method

no params

The load method performs the process of loading the sound file into the application, this process should only have to occur one time and then play many.

 
  load: ->
    try        
      s = new air.Sound()
      s.addEventListener air.Event.COMPLETE, (event) => 
        @set('sound', event.target)
        event.target.play()     
      req = new air.URLRequest @get('name')
      s.load(req)

    catch error
      air.trace error
